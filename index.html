Parichay (Introduction)

DhaakadScript ek aisi programming language hai jo seedha dil se nikalti hai. Ye JavaScript ke upar bani hai, par iske keywords aur andaaz poora Haryanvi hai.

Kyun banayi ye?

Coding seekhna kabhi-kabhi boring ho jata hai. DhaakadScript ka maksad hai coding ko mazedaar aur apni bhasha mein feel karwana.

Pehla Kadam

Sabse pehle, screen pe kuch likh ke dekhte hain.

Hello World (Ram Ram)

Kisi bhi cheez ko print karne ke liye bol keyword ka use karein.

bol("Ram Ram Saareyan Ne!");

Output:

> Ram Ram Saareyan Ne!

Bol (Print)

Jo muh mein hai, wo terminal pe bol de. Debug ya dialogue ‚Äì sab yahan se.

Maal-Paani (Variables)

Naya variable thap de (declare kar de). Naam de aur value chipka de.

‚ÄúJab bhi kuch yaad rakhna ho ‚Äì naam, umar, marks ‚Äì sab ko thap_le se shuru kara.‚Äù

Syntax

thap_le naam = "Raju"; thap_le umar = 25; thap_le pass_hai = sahi;

Booleans (Sahi / Galat)

Haryanvi true/false. Jo baat pakki ho, wo sahi. Jo jhooth ya nakli ho, wo galat.

Hisab-Kitab (Operators)

Jod-ghata aur tulna karne ke liye operators chahiye.

Ganit (Math)

+, -, *, / sab waise hi kaam karte hain.

thap_le x = 10 + 5; // 15 thap_le y = x * 2; // 30

Tulna (Comparison)

>, <, ==, != use karein.

agar (umar >= 18) { bol("Vote daal le bhai"); }

Rasta (Control Flow)

Condition ke hisaab se decision lene ka jugad. Pehle agar, fir zarurat ho to aur_agar, warna last option na_ta.

Agar - Magar

thap_le paise = 100; agar (paise > 500) { bol("Party karenge!"); } aur_agar (paise > 50) { bol("Chai piyenge."); } na_ta { bol("Ghar baithenge."); }

Geda/Chakkar (Loops)

Jab Tak (While Loop)

Jab tak condition sahi rahe, utni der chakkar lagata rahe.

thap_le petrol = 5; jab_tak (petrol > 0) { bol("Gaddi chal rahi hai..."); petrol = petrol - 1; }

Kitni Baar (For Loop)

Seedha counting loop ‚Äì kitni baar ghoomna hai, pehle hi bata de.

kitni_baar (i = 0; i < 5; i++) { bol("Round number: " + i); }

Kaam-Dhanda (Functions)

Reuse wali cheez. Ek baar likh, kaam ka naam rakh, jahan chahe baar-baar bula. Answer bhejne ke liye wapas_de laga.

Function Banana

kaam jod(a, b) { wapas_de a + b; } thap_le total = jod(10, 20); bol("Total hua: " + total);

DSA Roadmap (Dhaakad Version)

Data Structures aur Algorithms seekhne ka desi tareeka.

1. Basics

Loops, Conditionals, aur Functions ka template.

// For Loop kitni_baar (i = 0; i < n; i++) { ... } // While Loop jab_tak (condition) { ... } // Function kaam kuch_kaam(p1, p2) { wapas_de result; }

2. Arrays & Searching

Linear aur Binary Search.

// Linear Search kaam dhund_le(arr, target) { kitni_baar (i = 0; i < arr.length; i++) { agar (arr[i] == target) { wapas_de i; } } wapas_de -1; }

3. Sorting

Bubble Sort aur Insertion Sort.

// Bubble Sort kaam bubble_sort(arr) { thap_le n = arr.length; kitni_baar (i = 0; i < n - 1; i++) { kitni_baar (j = 0; j < n - 1 - i; j++) { agar (arr[j] > arr[j + 1]) { thap_le temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } } } wapas_de arr; }

Aur bhi topics hain: Recursion, Stack/Queue, Trees, Graphs, DP. Playground mein "Examples" dropdown se code load karke dekho!
