## Features
- **Desi Syntax**: Use keywords like `thap_le`, `bol`, `agar`, `jab_tak`.
- **Haryanvi Roasts**: Get roasted by the terminal if your code fails, or praised if it runs like a tractor.
- **Client-Side Execution**: Everything happens in your browser using a safe sandbox.
- **Responsive Design**: Works on desktop and mobile.

## How to Run
1. **Locally**:
   - Simply open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
   - No server needed!

2. **Deploy**:
   - Drag and drop this folder onto [Netlify Drop](https://app.netlify.com/drop) or upload to GitHub Pages / Vercel.
   - It's a static site, so it works anywhere.
